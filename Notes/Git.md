# 集中式与分布式

Git是分布式版本控制系统，SVN是集中式版本控制系统。

集中式版本控制系统：版本库集中存放在中央服务器，工作开始时使用本地电脑从中央服务器取得最新的版本，工作结束时再将本地电脑的版本送回中央服务器。

分布式版本控制系统：没有中央服务器，每个人的电脑都有一个完整的版本库
