# 三、持久化
Redis 是内存数据库，它将自己的数据库状态存储在内存中，为了保证数据库状态在断电后不会丢失，需要将内存中的数据持久化到硬盘中。
数据库状态：服务器中的非空数据库以及它们的键值对。
## RDB 持久化
RDB 持久化：将某个时间点的数据库状态保存到一个RDB文件
